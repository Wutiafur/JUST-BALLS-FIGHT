# Just Ball Game（v0.9）

一个基于 Pygame 的极简弹球乱斗小游戏：多实体自动移动、随机职业与道具、环境危机机制，支持在菜单中自定义地图与人数。

## 运行环境
- Python 3.8+（建议）
- Pygame

安装依赖：
```
pip install pygame
```

启动游戏：
```
python "game v0.9.py"
```

## 玩法简介
- 实体（球/方块）在地图内弹跳并互相碰撞。
- 会随机分配职业与道具，触发不同技能与效果。
- 可选择环境危机（毒圈、电墙、雷霆、过载或随机）。
- 计时结束或仅剩一名存活时结束。

## 操作说明
- 鼠标左键：菜单与设置项点击
- ESC：
  - 游戏中：暂停 / 继续
  - 设置界面、图鉴界面：返回菜单
- SPACE：结算界面返回菜单

## 设置项
在“开始游戏”前可选择：
- 地图：方形 / 宽屏 / 圆形
- 人数：2–8
- 时长：30秒 / 45秒 / 60秒 / 无限
- 环境：无 / 毒圈 / 电墙 / 雷霆 / 过载 / 随机

## 常见问题
- 界面中文字显示异常：请确认系统字体可用，或保持默认字体配置。
- 无法运行：检查 Python 与 Pygame 安装是否成功。
